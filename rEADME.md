# SpeedLoan  速來借

參與人員：徐子崴、徐躍華、王瀚

此為一個貸款需求媒合平台，分為兩種身份(代書端、用戶端)，由用戶端提出需求，再由代書來進行媒合。

使用資料庫為firebase，使用框架為React Native

用戶端
進行登入之後，填寫完個人資料，就可以提出申請需求。
如果需求已經被媒合，用戶端可以將需求標註為已解決

代書端
進行登入之後，填寫完事務所資訊之後，即可開始進行媒合。
